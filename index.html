Algorithmic Trading & Backtesting Platform

Python · FastAPI · PyTorch · scikit-learn · pandas · yfinance · pytest  |  Mar 2026 – Present

2 ML Strategies
36 Passing Tests
4 Architecture Layers
5 Performance Metrics

A production-quality algorithmic trading and backtesting platform following a strict layered monolith architecture (HTTP to service to domain to infra) with enforced unidirectional dependency flow, keeping the simulation engine independently testable and strategies hot-swappable.

The backtesting engine runs a bar-by-bar simulation loop, ingesting live OHLCV data via yfinance, applying configurable fees and slippage in basis points, and computing real performance metrics: total return, CAGR, Sharpe ratio, max drawdown, and win rate.

Two trained ML strategies are integrated: a RandomForest classifier on gold futures (GC=F) and a stateful Deep Q-Network agent on SPY with a 10-bar lookback buffer. Each strategy owns its own feature pipeline and is registered via a graceful-skip pattern that allows the app to start regardless of artifact availability.

The test suite uses a mock MarketDataPort that eliminates all network calls, validating happy-path and error-path behavior end-to-end across 7 test files.
